Sobre Nosotros


Proyecto de clase para la asignatura de entorno Cliente usando Angular en su versión 17 con componentes standalone y bootstrap para hacer una aplicación estilizada e intuitiva. La aplicación utiliza la API RestCountries para proporcionar información detallada sobre países de todo el mundo.

He desarrollado esta aplicación con el objetivo de brindar a los usuarios una experiencia intuitiva y educativa sobre la diversidad cultural, económica y geográfica de nuestro planeta.

¡Explora nuestro sitio y descubre fascinantes datos sobre países de todos los continentes!

Aplicación desarrollada por Natalia Zg.
